⚙️ Firebase Setup Required

Before using this app, connect it to a free Firebase project:

  1. Go to console.firebase.google.com → create a project
  2. Build → Realtime Database → Create database (test mode)
  3. Project Settings (gear icon) → scroll to "Your apps" → web icon → register → copy config
  4. Paste your config values into the firebaseConfig object in this file
  5. Also set your ADMIN_PASSWORD

Then open the file in a browser and share the link with friends (host on Netlify Drop for a shareable URL).
